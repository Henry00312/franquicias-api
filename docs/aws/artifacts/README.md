# Artefactos de despliegue (AWS)

Esta carpeta contiene JSON generados durante despliegues manuales con AWS CLI.

Archivos incluidos:

- `container-definitions.json`
- `ecs-register-td.json`
- `taskdef-aws.json`
- `taskdef-register.json`

Estos archivos se guardan solo como referencia operativa para troubleshooting y auditoría rápida.

No son requeridos para ejecutar la API localmente.
